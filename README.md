# chess_opening_classifier

## Team Members: Kannan Jain & Rachana Raju

### Overview:
Our goal is to build a model that will take in feature of the game after 10 full moves have been played and predict the opening that was played.

### Running the model
To run the model run the following files in this order
1. opening_predictor/prep/opening_family_games.py
2. opening_predictor/prep/popular_opening_games.py
3. opening_predictor/prep/15_fen_popular.py
4. opening_predictor/prep/split.py
5. opening_predictor/features.py
6. opening_predictor/train_model.py
